Corporal

A map app for giving or receiving aid locally by feeding the hungry, sheltering the homeless, and clothing the naked. | March 14th, 2026

Project Overview

Corporal is a 100% free and non-profit app on the iOS App Store. Corporal helps connect people to nearby donation bins, food pantries, and shelters that are often missing from major map apps. By making it easier to give and easier to get help, the project supports the Corporal Works of Mercy — feeding the hungry, clothing the naked, and sheltering the homeless. Corporal allows technology to become a tool for compassion, visibility, and community service through community submitted map-based aid locations. Any user can submit locations for all to serve or be served at.

I built Corporal as a part of my Ignatian Service Learning Justice Seminar class at Loyola Academy High School in the Chicagoland area. Corporal is built with JavaScript generally (although more specifically TSX) with the Expo Go React Native framework. I then set up version control with Git connected to a GitHub repository. My Resend SMTP server, Supabase database, and Railway server all interact with the code on this GitHub repository to ultimately build Corporal.

As of March 14th, 2026, Corporal's database has around 1000+ data points and 150+ community-submitted locations on the app across the United States.
